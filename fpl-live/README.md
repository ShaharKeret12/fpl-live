# FPL — Rice vs Sarr (Live)

פרויקט Next.js מוכן ל־Vercel עם פרוקסי ל־FPL API, שני אריחים להשוואת שחקנים ונקודות לייב.

## התקנה מהירה
```bash
npx create-next-app@latest fpl-live --ts --eslint --app --src-dir --tailwind --import-alias "@/*"
cd fpl-live
# העתק את הקבצים מהזיפ לתוך הפרויקט (שמור על מבנה התיקיות)
npm run dev
```

## דיפלוי ל־Vercel
חבר רפו ל־Vercel (או השתמש ב־Vercel CLI) ודחוף את הקוד.
